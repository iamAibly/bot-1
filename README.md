# Telegram Activation Bot

Бот для активации пользователей через подписку на каналы.

## 📦 Установка на BotHost.ru

### Способ 1: Через Docker (рекомендуется)

1. Загрузите все файлы в папку проекта
2. В настройках BotHost.ru укажите:
   - Тип запуска: Docker
   - Dockerfile: Dockerfile
   - Переменные: BOT_TOKEN=ваш_токен
3. Нажмите "Запустить"

### Способ 2: Через Python

1. Установите зависимости: `pip install -r requirements.txt`
2. Создайте .env файл с BOT_TOKEN
3. Запустите: `python bot.py`

## 🔧 Настройка

### Первый запуск
1. Отправьте боту команду `/start`
2. В админ-панели установите ID админа
3. Добавьте спонсоров

### Команды для админа
- `/admin` - открыть админ-панель
- `/start` - главное меню
